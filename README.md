# 📝 Minimalist Blog System

A clean, content-focused blog application built with **React**, powered by **Vite** and styled with **Tailwind CSS**. Inspired by modern content platforms, this project demonstrates professional frontend development practices with a strong focus on **typography**, **UX**, and **clean architecture**.

🌐 **Live Demo**: [https://minimalist-blog-system.vercel.app](https://minimalist-blog-system.vercel.app)

![Minimalist Blog Preview](https://images.unsplash.com/photo-1484480974693-6ca0a78fb36b?w=800&h=400&fit=crop&crop=center)

---

## ✨ Features

### 🎨 Design & UX
- Minimalist, single-column layout with generous whitespace
- Clean typography using Google Fonts (Inter & Playfair Display)
- Responsive design for all devices
- Elegant blurred snippet previews
- Subtle animations for interactions

### 📝 Content Management
- Create, edit, and view blog posts
- Posts support title, content, image, author, category, and tags
- Author and category dropdowns in the form
- Tag entry via comma-separated input
- In-memory state for simplified demo (no backend)

### 🔍 Discovery & Filtering
- Category filter on the homepage
- Tag-based filtering across pages
- Clickable categories/tags from post view
- Filters reflected in the URL for sharability

### ⚙️ Technical Stack
- **React 19** with Hooks and functional components
- **React Router** for SPA routing
- **Tailwind CSS** for rapid utility-first styling
- **Vite** for blazing-fast build and dev environment
- **Local JSON** as a dummy data source
